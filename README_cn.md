# OSN-ClientDemo
**作者：**  
`OSN-ClientDemo` 是一个IM客户端的演示程序。该工程基于android开发，主要演示了如何生成账号、如何添加好友、如何发送消息以及如何对消息进行封装和加密。OSN-ClientDemo目前只提供了最基础的演示，后续将持续更新。
